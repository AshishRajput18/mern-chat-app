# MERN Chat App

A full-stack real-time chat application built using the MERN Stack with secure authentication and real-time messaging functionality.

## 🚀 Features

* User Authentication (JWT)
* Real-time Messaging using Socket.IO
* Responsive UI Design
* Online/Offline User Status
* Secure Password Hashing
* MongoDB Database Integration
* Modern UI with Tailwind CSS & DaisyUI

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Tailwind CSS
* DaisyUI

### Backend

* Node.js
* Express.js

### Database

* MongoDB

### Other Technologies

* Socket.IO
* JWT Authentication
* bcryptjs
* Cookie Parser

---

## 📂 Project Setup

### Clone Repository

```bash
git clone https://github.com/AshishRajput18/mern-chat-app.git
```

### Install Dependencies

#### Backend

```bash
cd Backend
npm install
```

#### Frontend

```bash
cd Frontend
npm install
```

---

## ▶️ Run Project

### Start Backend

```bash
npm run server
```

### Start Frontend

```bash
npm run dev
```

---

## 🌐 Environment Variables

Create a `.env` file in Backend folder and add:

```env
PORT=5000
MONGO_DB_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
NODE_ENV=development
```
